Search & RAG
Test search and retrieval-augmented generation against indexed files.

          

            
Semantic
Semantic Search
Pure vector similarity using cosine distance on embeddings. Best for conceptual queries where meaning matters more than exact wording.

            
Hybrid (Default)
Hybrid Search (Recommended)
Combines vector similarity with BM25 keyword matching via Reciprocal Rank Fusion. Best default for mixed queries with concepts and specific terms.

            
Find Similar
Find Similar
Find files similar to a known file by vector proximity. Enter a file_id to discover related content.

            
RAG (Ask)
Retrieval-Augmented Generation
Ask a question, get a synthesized natural-language answer with source citations. Uses hybrid search + LLM.
